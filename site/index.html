Deployment Stack

Personal website

Static HTML and local avatar assets are built into an nginx:alpine container using this project's Dockerfile.

Todo application

The stack builds the Docker official sample dockersamples/todo-list-app source as the second application in the same docker-compose.yml deployment.

Automation

GitHub Actions copies the repository to the server, starts both services with Docker Compose, and verifies both URLs.

Server

Both applications are deployed on one server: 1.94.218.30.
